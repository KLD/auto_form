# auto_form

Create easy Flutter form pages without using controller.

## Code Example

## Supported fields

- AutoTextField
- AutoDropdownField
- AutoDateField
- AutoTimeField
- AutoFileField
- AutoCheckboxField
- AutoGroupField
- AutoComputedField

## Features

- Build Controlleress fields
- Triggers fields based on conditions
  - Hide/Show
  - Enable/Disbale
  - Show error on another field
  - Show error on form
  - Clear value
  - Set value
